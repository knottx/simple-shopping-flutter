# Simple Shopping

## API-Server

### To install dependencies

```bash
cd api-server-0.1.0

bun install
```

### To run

```bash
cd api-server-0.1.0

bun run index.ts
```

## Run App

```bash
flutter pub get
```

And Run directly from **VSCode**.
